# CarnageEngine
**"The Heart of Industrial Action."**

![Status](https://img.shields.io/badge/development-active-orange)
![Version](https://img.shields.io/badge/version-0.1.0-blue)
![License](https://img.shields.io/badge/proprietary-internal-blue)

CarnageEngine is the proprietary, high-performance game engine powering *Through the Eyes of Carnage*. Designed specifically for high-intensity, industrial-themed action, the engine focuses on visceral impact, cybernetic aesthetics, and mechanical precision.

---

## 🛠 Core Feature-Set

| System | Description | Status |
| :--- | :--- | :--- |
| **Physics** | Powered by [CiPhyxs](https://github.com/DriveBy-Studios/CiPhyxs) | Planned |
| **Audio** | Industrial soundscape & procedural cyber-drone synthesis | Planned |
| **Rendering** | High-contrast, gritty, industrial-grade visuals | In-Progress |
| **AI** | Mechanical enemy behavior & cyborg pathfinding | Planned |

## ⚙️ Technical Foundation
CarnageEngine is built upon industry-standard, high-performance libraries:

* **Graphics API:** [Vulkan](https://www.vulkan.org/)
* **Windowing/Input:** [GLFW](https://www.glfw.org/)
* **Math Library:** [GLM](https://glm.g-truc.net/)
* **UI/Editor:** [Dear ImGui](https://github.com/ocornut/imgui)
* **Audio Engine:** [MiniAudio](https://miniaud.readthedocs.io/)
* **Memory Management:** [Vulkan Memory Allocator (VMA)](https://github.com/GPUOpen-LibrariesAndSDKs/VulkanMemoryAllocator)
* **Logging:** [spdlog](https://github.com/gabime/spdlog)
* **Asset Loading:** [Tiny OBJ Loader](https://github.com/tinyobjloader/tinyobjloader)

---

## ⚙️ Philosophy: "The Law of Carnage"
CarnageEngine is built on the belief that physics should feel like a governing law rather than a background calculation. Every impact, stomp, and movement is designed to feel heavy, industrial, and deliberate.

## 🔗 Integrated Ecosystem
We believe in contributing back to the developer community. While CarnageEngine is proprietary, its core physics module is open-source and available to the public:
* **[CiPhyxs](https://github.com/DriveBy-Studios/CiPhyxs):** The official physics solver for CarnageEngine.
