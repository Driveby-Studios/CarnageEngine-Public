# Contributing to CarnageEngine

We appreciate your interest in CarnageEngine! While the engine source code is proprietary and closed-source, we welcome community engagement through this public repository.

## How to Contribute
You can help us improve the engine's public-facing documentation and roadmap:

1. **Suggest Features:** Open a [Discussion](https://github.com/DriveBy-Studios/CarnageEngine-Public/discussions) to propose new features or improvements for the engine's public roadmap.
2. **Report Documentation Issues:** If you find errors or ambiguities in our public documentation, please open an issue.
3. **Engage with CiPhyxs:** Since our physics module is open-source, we actively welcome pull requests and issues on the [CiPhyxs repository](https://github.com/DriveBy-Studios/CiPhyxs).

## Code of Conduct
Please maintain a professional and constructive tone in all discussions and issues. We are building a high-performance, industrial-grade tool, and we appreciate your respect for the development process.
